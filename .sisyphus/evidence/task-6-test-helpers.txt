TOPOLOGY HELPERS USAGE:
=======================

File: tests/helpers/topology-helpers.js

1. EPS = 1e-6
   - Floating-point tolerance for distance comparisons
   - Used by all assertion helpers

2. assertContinuity(segments)
   - Validates that segment[i].end connects to segment[i+1].start
   - Throws error if gap >= EPS
   - Provides detailed error messages with point coordinates and gap distance

3. assertNoZeroLength(segments)
   - Validates all segments have distance(start, end) >= EPS
   - Throws error if any segment has length < EPS
   - Prevents degenerate/parasitic segments

4. assertContourCount(result, expected)
   - Simple assertion for exact contour count match
   - Throws error with actual count if mismatch

    it("single-line offset should have continuous segments", async () => {
        const engine = new OffsetEngine({ joinType: "sharp", capType: "flat", exportModule });
        const pathData = "M 0 0 V 10";
        const result = await engine.processPath(pathData, 1);

        assertContourCount(result, 1);
        const contour = result.contours[0];

        // Verify continuity
        assertContinuity(contour.segments);

        // Verify no zero-length segments
        assertNoZeroLength(contour.segments);
    });
