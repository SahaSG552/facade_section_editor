**Implementation Details**:
1. `trimSelfIntersections(segments)`: 
   - Converts input segments to SVG path via `segmentsToPathString()`
   - Delegates to `PaperBooleanProcessor.resolveSelfIntersections()` for trimming
   - Parses result back to segment array via `pathStringToSegments()`
   - Returns clean segments, empty array on error

2. `segmentsToPathString(segments)`:
   - Handles line, arc (both formats), and bezier segments
   - Generates SVG path commands (M, L, A, C, Z)
   - Auto-closes path if start ≈ end (0.001 tolerance)
   - Returns valid SVG d attribute string

3. `pathStringToSegments(pathString)`:
   - Lightweight SVG parser for absolute commands
   - Reconstructs segment objects from path data
   - Suitable for round-trip: segments → SVG → segments
   - For complex relative paths, can use ExportModule parser
